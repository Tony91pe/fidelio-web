# Legal Documentation - Fidelio

This directory contains all required legal texts and notices.

## Files

1. **PRIVACY_POLICY.md** - GDPR-compliant privacy notice
2. **TERMS_OF_SERVICE.md** - General terms and conditions
3. **COOKIE_POLICY.md** - Cookie usage and consent
4. **GDPR_NOTICE.md** - GDPR information notice
5. **DPA.md** - Data Processing Agreement for processors
6. **APP_STORE_TEXTS.md** - App Store and Play Store descriptions
7. **EMAIL_TEMPLATES.md** - Legal texts for email communications
8. **COOKIE_BANNER.md** - Cookie consent banner copy
9. **ONBOARDING_LEGAL.md** - Onboarding flow legal texts
10. **README.md** - This file

## Usage

- Display privacy policy and terms on sign-up
- Include DPA with merchant contracts
- Use email templates for automated communications
- Display cookie banner on first visit
- Include GDPR notice in app and web
- Update app store listings with provided texts

## Customization

Replace [placeholders] with actual information:
- [Country], [City]
- [Email], [Address]
- [Local DPA] reference

## Updates

Review all documents annually for legal changes.
Keep git history for compliance documentation.
